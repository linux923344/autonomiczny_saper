# Instalacja
Dodatkowo trzeba posiadać:
* Python 3.7
* Vowpal Wabbit

Żeby odpalić program trzeba zainstalować następujące biblioteki:

```bash
pip install pygame
pip install inflect
pip install sklearn 
pip install joblib
pip install graphviz
pip install pydotplus
```
